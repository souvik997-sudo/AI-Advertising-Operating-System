# Script Framework Module

Version: 1.0

---

# Purpose

This module defines how the AI should generate high-converting advertising scripts.

The objective is to create scripts that sound completely natural, emotional, conversational and human-written.

The AI must never sound robotic.

---

# Script Creation Workflow

1. Understand the Brand

2. Understand the Target Audience

3. Identify Customer Pain Points

4. Identify Customer Desire

5. Select the Marketing Angle

6. Select Emotional Trigger

7. Create Hook

8. Build Curiosity

9. Present the Problem

10. Present the Solution

11. Explain the Benefits

12. Add Social Proof

13. Add Emotional Connection

14. Create Strong CTA

15. Review

16. Humanize

---

# Script Structure

## Hook

Capture attention immediately.

---

## Curiosity

Make the audience want to continue watching.

---

## Problem

Explain the customer's pain.

---

## Solution

Introduce the brand naturally.

---

## Benefits

Explain customer benefits.

Never list features only.

---

## Emotional Connection

Make the audience feel something.

---

## Social Proof

Increase trust.

---

## Call To Action

Simple

Natural

Direct

---

# Human Writing Rules

Always write like a real person.

Never sound like AI.

Never repeat sentence patterns.

Use natural conversation.

Use emotion.

Use pauses.

Use rhythm.

Use storytelling.
