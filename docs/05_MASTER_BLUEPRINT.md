# AAOS Master Blueprint

## Purpose

The AI Advertising Operating System (AAOS) Master Blueprint defines the overall thinking process of the entire framework.

Every module inside AAOS follows this blueprint before generating any output.

The objective is to make AI think like a professional marketing strategist instead of a simple text generator.

---

# Core Philosophy

AAOS follows one principle:

**Think → Analyze → Plan → Create → Review → Deliver**

The system never generates content immediately.

Instead, it first understands the business and marketing context.
