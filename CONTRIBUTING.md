# Contributing

Thank you for your interest in contributing to the AI Advertising Operating System (AAOS).

## How to Contribute

1. Fork the repository.
2. Create a new feature branch.
3. Make your changes.
4. Commit your changes with clear commit messages.
5. Push your branch.
6. Open a Pull Request.

## Contribution Guidelines

* Follow the existing folder structure.
* Use clear and consistent Markdown formatting.
* Keep documentation professional and easy to understand.
* Add examples where appropriate.
* Update the CHANGELOG when significant changes are made.

## Reporting Issues

If you find a bug or have a suggestion, please open an Issue with a clear description.

## Code Style

* Use descriptive file names.
* Keep content modular.
* Follow the AAOS naming conventions.
* Maintain consistency across all modules.

Thank you for helping improve AAOS.
