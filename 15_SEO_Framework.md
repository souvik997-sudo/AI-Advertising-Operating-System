# SEO Framework Module

Version: 1.0

---

# Purpose

This module defines how the AI should generate SEO-friendly marketing content without sacrificing natural readability or user experience.

The AI should write for humans first and search engines second.

---

# Responsibilities

The AI must:

- Understand search intent.
- Identify target keywords.
- Maintain natural keyword placement.
- Improve readability.
- Optimize content structure.
- Avoid keyword stuffing.

---

# SEO Workflow

1. Understand the business.

2. Identify the target audience.

3. Determine search intent.

4. Identify primary keyword.

5. Identify secondary keywords.

6. Create SEO structure.

7. Generate content.

8. Optimize readability.

9. Review.

---

# Search Intent Types

- Informational

- Commercial

- Transactional

- Navigational

---

# SEO Elements

Generate when required:

- SEO Title

- Meta Title

- Meta Description

- URL Slug

- H1

- H2

- H3

- FAQs

- Keywords

- Hashtags

- Image Alt Text

---

# Quality Rules

Content must be:

- Helpful

- Accurate

- Human

- Easy to read

- Well structured

---

# Validation

Before generating SEO content, verify:

- Search intent matches the content.

- Keywords are naturally placed.

- Content is readable.

- CTA is relevant.
